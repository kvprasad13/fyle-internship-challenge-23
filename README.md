# Using the Application

  Clone the repository and install the dependencies as given below.

  
  ## In Client

  ```
  cd client
  npm install
  npm install -D tailwindcss
  npx tailwindcss init
  
  
  ```

  ### Start Client

  ```
  npm start
  ```


  ## In Server

  ```
  cd server
  npm install

  
  ```
   ### Start Server
   ```
  npm start
   ```
